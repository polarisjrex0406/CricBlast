## How to Run

- **Standalone Executable:** Does not require any configuration. Simply download the [`CricBlast.exe`](https://github.com/Raofin/CricBlast/releases/download/v0.1.0/CricBlast.exe) and run!
- [**Source Code**](https://github.com/Raofin/CricBlast/archive/refs/tags/v0.1.0.zip): Before running the program, please complete the following two steps:
  1. Import the database backup ([`.bacpac`](https://github.com/Raofin/CricBlast/releases/download/v0.1.0/CricBlast.bacpac)) file into **Microsoft SQL Server**.
     <img src="https://imgur.com/tVoggH9.png" title="" alt="" width="690">
  2. Enter your server name into `App.config`.
     <img src="https://imgur.com/ygLeNbr.png" title="" alt="" width="690">
